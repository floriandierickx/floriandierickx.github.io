Country Emission Budget Calculator

The purpose of below application is to create an interactive version of the Emission Budget blogpost of Stefan Rahmstorf . Currently it displays historical country emissions, and calculates both global and country-specific carbon budgets based on a given total carbon budget. The goal is to also implement future emission budgets, linearly decreasing from 2020 onwards. Help on this is welcome on GitHub [see the README.md and Issue], where you can find more details on how the code is structured.
